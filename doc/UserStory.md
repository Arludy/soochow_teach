# 用户故事

- 作为用户中的老师个体，我想要通过认证以及填写完具体信息后，能查看学生资源页面中距离近，薪资合适的学生个体，进而跟他们在平台上私聊，商谈进一步的家教细则。
同时，我也可以在填写完基本信息后，同意将自己的信息发布在学生找老师的页面上，这样倘若有学生对我感兴趣，我可以在消息栏中看到他们发来的私聊信息。
这样，我可以又快又好的找到合适的学生资源。

- 作为用户中的学生个体，我想要通过平台搜寻合适的老师，通过筛选时间、距离、科目等功能，进行筛选老师，对感兴趣的老师进行私聊。
同时，我也可以填写自己的基本信息，来让自己的信息显示在老师找学生的页面上，然后再消息栏中，找到对我感兴趣的老师的私聊信息。
这样，我可以又快又好的找到合适的老师资源。

- 作为用户，我们的言行也被平台监管者，这样，学生和老师的交流可以有安全保障。
